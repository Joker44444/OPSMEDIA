# OpsMedia × MEDIACRAFT — Landing Page

> 문의가 들어오는 순간, 기획은 자동으로 시작됩니다.

OpsMedia의 AI 영상 기획 솔루션 **MEDIACRAFT** 소개 랜딩페이지입니다.

## 🌐 라이브 사이트

배포되면 아래 URL에서 확인할 수 있습니다.

```
https://[GitHub 유저명].github.io/[리포지토리명]/
```

## 📂 구성

- `index.html` — 단일 HTML 파일 (CSS·JS 모두 인라인 포함)
- 외부 의존성: Google Fonts (Gowun Batang)

## 🎨 디자인 노트

- **단일 폰트** — Gowun Batang 한 가지로 통일, weight·size로만 위계 표현
- **흑백 페이퍼 톤** — 종이 그레인 텍스처 + `#f4f1ea` 페이퍼 컬러
- **관통 디자인 어셋** — 좌측 세로 가이드 라인 · 스크롤 진행 도트 · 섹션 인덱스(§00–§07) · 고양이 발자국 모티프
- **반응형** — 데스크탑 / 태블릿 / 모바일 대응

## 🛠 로컬에서 확인

별도 빌드 없이 브라우저로 `index.html`을 열기만 하면 됩니다.
